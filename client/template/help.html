1. BBCode Tag Usage

1.1 Basic Markup

Whether or not a tag may be nested inside other tags depends on the tags. Generally, placing other tags inside a tag that greatly alters the input text, such as [rainbow], [gradient], or [spoiler], will not work correctly. Placing other tags inside of [i], [b], or [u] tags usually works.

Tag Result
[i]italic text[/i] italic text
[b]bold text[/b] bold text
[u]underlined text[/u] underlined text
[url=http://example.org/]Click here![/url] Click here!
[rainbow]rainbow text[/rainbow] rainbow text
[gradient start=yellow end=blue]gradient text[/gradient] gradient text
[blink]blinking text[/blink] blinking text
[marquee]scrolling text[/marquee]
scrolling text

1.2 The [img] Tag

This tag may only be used with images that you or other members have uploaded. Placing the URL of an image from another website inside the [img] tag will not work. When you upload a file, you'll see a page of information about the file. One of the items shown is a 32-character MD5 hash of the file. Copy and paste the MD5 hash value between the [img] tags.

Tag Result
[img]11e21d7da3d9d8d679e116233aa8255a[/img]

1.3 The [spoiler] Tag

This tag scrambles text. Click the Decode button to unscramble it.

Tag Result
[spoiler]Han Solo shot first.[/spoiler]
Spoiler:
Una Fbyb fubg svefg.

1.4 The [code] Tag

This tag can be used with or without syntax highlighting. For syntax highlighting, specify the programming language in the opening tag. For ASCII art or command line input and output, omit the programming language. In the examples below, we display the same JavaScript code with and without syntax highlighting.

Tag Result 
[code=javascript]
function sleep(milliseconds)
{
  var start = new Date().getTime();
  for (var i = 0; i < Number.MAX_SAFE_INTEGER; i++) {
    if ((new Date().getTime() - start) > milliseconds) {
      break;
    }
  }
} // sleep

1.5 The [youtube] Tag

Place the video ID between the tags. For example, if you have this YouTube URL:

https://www.youtube.com/watch?v=gqc8AqIZj4w

you would copy and paste the gqc8AqIZj4w part between the tags.

Tag Result
[youtube]gqc8AqIZj4w[/youtube]

1.6 The [ansi] Tag

As with the [img] tag, the [ansi] tag will only work with ANSI art files you or other members have uploaded. Place the file's MD5 hash value between the tags.

NOTE: This is a preliminary implementation. The tag currently assumes the file is an 80-column Code Page 437 file. This tag will soon change to allow users to specify other character sets and column widths.

The following example uses a PNG image to simulate the result. The actual result will be a full-size HTML rendering of the Extended ASCII characters in your file.

Tag Result
[ansi]ff5d8fbfbb15323fc9d1d9bc538a1cfd[/ansi]
